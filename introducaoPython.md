Análise de dados da saúde compõe um importante pilar para a modernização, otimização e transformação geral dos sistemas de saúde. Trata-se do processo de coleta, refinamento e interpretação de grandes volumes de informação geradas por prontuários online, resultados de exames, dados genômicos, dispositivos vestíveis (`wearables`) e registros administrativos.  

Para que a análise de dados seja feita de forma correta, é preciso escolher bem as ferramentas para tal. Existem muitas opções para isso, mas utilizaremos a linguagem de programação Python, suas bibliotecas especializadas e `frameworks` (um framework pode ser entendido como uma estrutura pré-construída ou um esqueleto que fornece uma base para o desenvolvimento de software ou, no nosso contexto, de um projeto de raspagem de dados), aproveitando o poder dessas ferramentas para transformar dados brutos em `insights` valiosos e acionáveis. 

## O que é Python

[Python](https://www.python.org/) é uma linguagem de [programação interpretada](https://www.freecodecamp.org/news/compiled-versus-interpreted-languages/) (programa executa linha por linha separadamente), de [alto nível](https://en.wikipedia.org/wiki/High-level_programming_language) (mais "humanizada"), e multiplataforma (roda em Windows, Linux, Ubuntu, etc...). Ela foi criada por [Guido Von Rossum](https://pt.wikipedia.org/wiki/Guido_van_Rossum) no início dos anos 1990.
A sintaxe do `Python` é simples e legível, o que a torna ideal tanto para programadores iniciantes quanto para profissionais experientes.
Com o passar dos anos, a linguagem  `Python` se consolidou como uma das linguagens mais utilizadas no mundo, e a partir de 2023, se tornou a [linguagem com mais programadores](https://www.tiobe.com/tiobe-index/), onde $24\%$ dos programadores sabem escrever nela, além disso com os [dados da jetbrains](https://lp.jetbrains.com/python-developers-survey-2022/), vemos que, em 2022, o `Python` é a linguagem principal de $85\%$ dos desenvolvedores, além disso `Python` é usado em diversas áreas, como analise de dados, desenvolvimento web, aprendizado de máquina, automação e acadêmica.

## Por que utilizar Python nesse contexto?

`Python` é uma linguagem muito versátil e possui muitas bibliotecas (conjunto de códigos pré-escritos que fornecem funcionalidades prontas para serem reutilizadas em diferentes projetos, economizando tempo e esforço do desenvolvedor) e `frameworks` (estrutura predefinida que fornece uma base de código, diretrizes e ferramentas para simplificar o desenvolvimento de projetos, especialmente em tecnologia) especializadas em análise de dados.
