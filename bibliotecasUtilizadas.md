## [Requests](https://requests.readthedocs.io/en/latest/user/quickstart) 
Requests permite que você envie solicitações HTTP/1.1 de forma extremamente fácil. Não há necessidade de adicionar manualmente strings (cadeia de caracteres) de consulta para seus URLs, ou codificar seus dados POST. Pooling (agrupamento) de conexões keep-alive e HTTP é 100% automático, graças ao urllib3 
 

## [BeautifulSoup](https://beautiful-soup-4.readthedocs.io/en/latest/#quick-start) 
Beautiful Soup  é uma biblioteca Python para obter dados de arquivos HTML e XML. Funciona com  seu analisador favorito provendo formas idiomáticas de navegação, procura, e modificação da árvore de análise sintática. Comumente otimiza horas e dias de trabalho dos programadores. 
 

## [Selenium](https://selenium-python.readthedocs.io/getting-started.html) 
Os bindings (ligações) do Selenium para Python fornecem uma API simples para escrever testes funcionais/de aceitação usando o Selenium WebDriver. Através da API do Selenium Python, você pode acessar todas as funcionalidades do Selenium WebDriver de uma forma intuitiva. 

Os bindings do Selenium Python fornecem uma API conveniente para acessar WebDrivers como Firefox, Ie, Chrome, Remote, etc. As versões atuais de Python suportadas são a 3.5 e superiores. 

 
## [LXML](https://lxml.de/tutorial.html) 
O toolkit XML lxml é uma ligação Pythonica para as bibliotecas C libxml2 e libxslt. É único porque combina a velocidade e a completeza de recursos XML dessas bibliotecas com a simplicidade de uma API Python nativa, amplamente compatível, mas superior à bem conhecida API ElementTree. A versão mais recente funciona com todas as versões CPython de 3.6 a 3.12. Consulte a introdução para obter mais informações sobre o contexto e os objetivos do projeto lxml. 


## [Pandas](https://pandas.pydata.org/docs/user_guide/index.html) 
pandas é um pacote Python que fornece estruturas de dados rápidas, flexíveis e expressivas, projetadas para tornar o trabalho com dados “relacionais” ou “rotulados” fácil e intuitivo. Seu objetivo é ser o bloco de construção fundamental de alto nível para realizar análises de dados práticas e do mundo real em Python. Além disso, tem o objetivo mais amplo de se tornar a ferramenta de manipulação/análise de dados de código aberto mais poderosa e flexível disponível em qualquer linguagem. 

 
## [NumPy](https://numpy.org/doc/2.3/user/quickstart.html) 
NumPy é o pacote fundamental para a computação científica em Python. É uma biblioteca Python que fornece um objeto de array multidimensional, vários objetos derivados (como arrays mascarados e matrizes) e um conjunto de rotinas para operações rápidas em arrays, incluindo matemática, lógica, manipulação de forma, classificação, seleção, E/S (I/O), transformadas de Fourier discretas, álgebra linear básica, operações estatísticas básicas, simulação aleatória e muito mais.

## [Scrapy](https://docs.scrapy.org/en/latest/intro/tutorial.html) 
Scrapy é um framework rápido de alto nível para web crawling (rastreamento) e web scraping (extração) de páginas da web, usado para rastrear sites e extrair dados estruturados de suas páginas. Pode ser usado para uma ampla gama de propósitos, desde mineração de dados até monitoramento e testes automatizados.
