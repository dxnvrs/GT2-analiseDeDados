## O que são variáveis?

Uma variável é um espaço na memória do computador destinado a um dado que é alterado durante a execução do algoritmo. Para funcionar corretamente, as variáveis precisam ser definidas por nomes e tipos. Em Python, diferente de outras linguagens como C ou Java, não é necessário declarar explicitamente o tipo da variável - ele é inferido automaticamente pelo interpretador.

## Tipos de dados

Python possui diversos tipos de dados incorporados que podem ser organizados nas seguintes categorias:

numéricos, texto, booleanos, coleções e None

### Numéricos
Nessa categoria estão os tipos `int`, `float` e `complex`. O tipo `int` representa números inteiros positivos ou negativos, sem parte decimal. O tipo `float` representa números reais, com parte decimal. O tipo `complex` é usado principalmente em cálculos matemáticos e engenharia.

### Texto
Nessa categoria está apenas o tipo `str` ou String. As strings em Python são coleções de caracteres delimitadas por aspas simples ou duplas. Podem ser manipuladas de várias formas.

### Booleanos
Nessa categoria está apenas o tipo `bool`. Os booleanos representam valores de verdade e podem ser apenas True (Verdadeiro) ou False (Falso).

### Coleções
Python oferece diversos tipos de coleções para armazenar múltiplos valores: lista `list`, tupla `tuple`, dicionário `dict` e conjunto `set`. O tipo `list` são coleções ordenadas e mutáveis (podem conter elementos de diferentes tipos). O tipo `tuple` são coleções ordenadas e imutáveis (não podem ser alteradas após criação). O tipo `dict` são coleções de pares chave-valor (as chaves devem ser únicas e imutáveis). O tipo `set` são coleções não ordenadas de itens únicos.

### None

O tipo None em Python representa a ausência de valor ou um valor nulo.
